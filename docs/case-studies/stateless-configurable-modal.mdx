## Stateless Configurable Modal

This demo showcases a stateless, configurable modal architecture using
derived availability logic and render prop injection.

---

<div className="demoSection">
  <h3>Default Behavior</h3>
  <EditProductModal />
</div>

<div className="demoSection">
  <h3>Injected CTA (Render Prop Override)</h3>
  <EditProductModal
    renderPrimaryCta={(context) => (
      <button
        className="primaryButton injectedCta"
        disabled={!context.isAvailable}
      >
        {context.isAvailable
          ? `Confirm ${context.selectedColor} / ${context.selectedSize}`
          : "Select a valid combination"}
      </button>
    )}
  />
</div>
